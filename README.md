Smart Study Flashcards & MCQ Generator

Project Overview
Smart Study Flashcards & MCQ Generator is a Python-based application that automatically converts PDF study materials into interactive flashcards and multiple-choice questions (MCQs). 
The system extracts text from uploaded PDF files, processes the content, and generates learning aids that help students quickly review concepts and test their understanding.
The project aims to improve learning efficiency by transforming lengthy study notes into concise flashcards and self-assessment questions.

Key Features
	Upload PDF study materials
	Automatic text extraction from PDFs
	Generation of flashcards for quick revision
	Rule-based MCQ generation from study content
	Interactive MCQs with answer checking and scoring
	Supports multiple PDF uploads

Technologies Used
	Python
	PyMuPDF (fitz) for PDF text extraction
	Regular Expressions (re) for text cleaning
	Streamlit for building the interactive web application
	
How It Works
	The user uploads a PDF document containing study material.
	The application extracts the text from the PDF.
	The extracted text is cleaned and divided into meaningful sentences.
	Flashcards are generated from key sentences in the document.
	Rule-based logic identifies important concepts and generates MCQs.
	Users can attempt MCQs interactively and view their scores.

Applications
	Quick revision for students
	Self-assessment for exam preparation
	Converting lecture notes into study tools
	Educational demonstrations

Future Improvements
Future versions of the project can include AI-based question generation, support for more document formats, adaptive difficulty levels for questions,
and exporting generated flashcards or quizzes as PDF files.
